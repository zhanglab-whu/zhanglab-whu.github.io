# Zhang Lab Website

Static website for Zhang Lab at the College of Life Sciences, Wuhan University.

## Overview

This site uses only plain HTML and CSS. It is designed to work directly with GitHub Pages at:

https://zhanglab-whu.github.io

## Files

- `index.html` - Home page with lab overview, profile image, methods, news, and open positions.
- `research.html` - Research themes, methods, representative progress, and future directions.
- `publications.html` - Selected publications grouped by corresponding-author and first-author papers.
- `people.html` - Principal Investigator profile and placeholder lab member sections.
- `contact.html` - Contact and open positions information.
- `style.css` - Shared responsive styling.
- `assets/zhang-xinghua-profile.png` - Public website image supplied for the lab site.

## Editing Notes

- Replace placeholder people, news, and contact details only with verified public information.
- Do not add unverified publications, awards, impact factors, private contact details, or tracking scripts.
- Keep links relative so the site remains compatible with GitHub Pages.
- Commit only public website files. Do not commit CVs, Word files, PDFs, passport or ID documents, contracts, application materials, or personal files.

## Deployment

Push these files to the root of the `zhanglab-whu.github.io` repository. GitHub Pages will serve the site from the repository root.
